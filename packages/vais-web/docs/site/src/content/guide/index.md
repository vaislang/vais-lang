# Guide

Welcome to the VaisX framework guide. This section covers everything you need
to know about building production-grade applications with VaisX.

## Overview

VaisX is structured around three core packages:

| Package | Purpose |
|---|---|
| `@vaisx/runtime` | Reactive primitive — signals, effects, DOM patching |
| `@vaisx/kit` | Router, SSR/SSG helpers, file-based conventions |
| `@vaisx/components` | Tree-shakeable UI primitives |
