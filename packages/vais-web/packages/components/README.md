# @vaisx/components

VaisX standard UI components — a set of accessible, zero-dependency primitives including Button, Input, Link, and Head.

## Installation

```bash
npm install @vaisx/components
```

## Usage

```ts
import { Button, Input, Link, Head } from '@vaisx/components'

// In a .vaisx template
// <Button variant="primary" onClick={handleClick}>Click me</Button>
// <Input type="text" value={name} onInput={setName} />
// <Link href="/about">About</Link>
// <Head title="My Page" />
```
